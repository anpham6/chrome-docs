Changelog

Added in version 0.11.0:

  • IHost property getters ipV4 | ipV6 for remote client address were created.

Added in version 0.10.0:

  • IClientDb method getCacheResult was created.

Changed in version 0.10.0:

  • IHost methods pauseLog | resumeLog argument type as string was implemented.

Added in version 0.9.0:

  • IHost methods were created:

    • pauseLog

    • resumeLog

    • hasLog

    • delayMessage

    • updateProgress

  • IHost property logState was created.

Removed in version 0.9.0:

  • IClientDb method hasCache argument override as DbCacheValue.

  • IClientDb method hasCoerce argument override as DbCoerceValue.

Example usage

Abstract class
const { Host } = require("@e-mc/core");

Host.loadSettings({ // Global
  process: {
    thread: { limit: 8 }
  },
  permission: {
    disk_read: ["**/*"],
    disk_write: ["/tmp/**"]
  }
});

Attention

@e-mc/core is mostly a collection of abstract base classes which cannot be instantiated. Host is more commonly called through @e-mc/file-manager.
