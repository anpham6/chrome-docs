Changelog

Added in version 0.10.0:

  • ModuleConstructor static property PLATFORM_WIN32 was created.

  • IModule property accessor silent for console messages was created.

Changed in version 0.10.0:

  • ModuleConstructor static method asHash argument minLength was replaced with digest as BinaryToTextEncoding.

Deprecated since version 0.10.0:

  • ModuleConstructor static method supported is a wrapper for Types.supported.

Added in version 0.9.0:

  • ModuleConstructor static property LOG_FORMAT was created.

Changed in version 0.9.0:

  • IModule method src and dest arguments can accept URL object:

    • canRead

    • canWrite

    • readFile

    • writeFile

    • deleteFile

    • copyFile

    • moveFile

    • createDir

    • removeDir

    • streamFile (static)

Added in version 0.8.7:

  • IModule method getPackageVersion optional argument baseDir as string was created.

Changelog

Added in version 0.11.0:

  • TempModule property os for system global temp directory was created.

Changed in version 0.11.0:

  • MemoryModule settings property cache_disk.exclude can be prefixed with “!” to negate a subset of glob paths.

Added in version 0.10.0:

  • LoggerModule group format block error for output display was created.

  • LoggerModule group format property braces for text separation was implemented.

  • LoggerModule group progress for summary data was created.

  • TempModule property env for system user local temp directory was was created.

Added in version 0.8.6:

  • PermissionModule properties home_read | home_write were implemented.
